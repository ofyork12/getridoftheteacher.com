The standard number pad — colored dots + digit, in every game. Tap the + / − to make it bigger or smaller.

  const kb = CMMKeyboard.mount('#myKeyboard', {
    onPress: n => addDigit(n),   // 0–9 pressed
    onErase: ()  => backspace(),  // erase pressed
    size: 64,                     // start size in px
    controls: true                // show bigger/smaller buttons
  });
