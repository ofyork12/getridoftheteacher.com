📚 The Hughes Method

A whole math curriculum built on one idea: a number is a color, drawn as dots. A child sees the math before reading a word — so it works in any language. Every skill moves the same way: See it → Count it → Mix it → Master it. Easy patterns first.

Created by Ted Hughes for children · hughesmath.net · Creative Commons (BY-NC)
