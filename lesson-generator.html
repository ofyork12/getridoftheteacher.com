🎨 Generate images on a separate AI

Pictures help learners remember sentences. Here's how to make them on another tool, then bring them back here:

  1. Open an image generator: ChatGPT (DALL·E), Google Gemini, Midjourney, Bing Image Creator, or Leonardo.ai (most have a free tier).
  2. For each sentence, ask for an image with a prompt like:
    A simple, friendly cartoon illustration of: "[your sentence here]". Bright colors, clear subject, no text in the image.
  3. Either: (a) download the image and use the Choose image button below each sentence, OR (b) right-click the generated image, copy the image address, and paste it into the Image URL field.
  4. Tip: uploaded images make a bigger HTML file but always work offline. URLs keep the file small but need internet to view.
